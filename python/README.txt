# Kumbh Mela AI Prototype

This is a **prototype project** for an AI-powered assistant for Kumbh Mela.

## 📦 Files
- `app.py` → Streamlit frontend (UI)
- `backend.py` → Flask backend (dummy answers)
- `requirements.txt` → Required Python libraries
- `README.txt` → Instructions

## ▶️ How to Run
1. Open PowerShell (or terminal) inside the folder.
2. Install requirements:
   ```bash
   pip install -r requirements.txt
   ```
3. Start backend:
   ```bash
   python backend.py
   ```
4. In another terminal, start frontend:
   ```bash
   streamlit run app.py
   ```
5. Browser will open → Ask questions.

⚠️ This is a **prototype**: Answers are static (demo only).
